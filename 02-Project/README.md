# 🎯 02-Project 项目库

> 正在进行的短期任务、项目、目标。用完归档。

## 使用规则

1. **一项目一文件夹** — 独立空间，不混杂
2. **模板驱动** — 使用 project-template 创建
3. **及时归档** — 完成后移入 98-Archive

## 项目结构

```
02-Project/
├── 项目名称/
│   ├── README.md        ← 项目概述
│   ├── 进度台账.md       ← 进度记录
│   ├── 问题记录.md       ← 遇到的问题
│   └── 成果归档.md       ← 最终成果
```
