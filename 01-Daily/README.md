# 📅 01-Daily 每日日志

> 日/周/月复盘、日程、待办。工作流核心枢纽。

## 使用规则

1. **每日一记** — 一个文件 = 一天
2. **模板驱动** — 使用 daily-template 创建
3. **链接串联** — 前后日期用 wikilink 连接

## 文件命名

```
YYYY-MM-DD 每日复盘.md
```

## 复盘节奏

- **每日**: 晚睡前 5 分钟，核对任务，记录心得
- **每周**: 周末 30 分钟，汇总收获，补全链接
- **每月**: 月末 1 小时，归档项目，优化结构
