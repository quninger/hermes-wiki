# 📚 03-Knowledge 知识库

> 常青笔记、永久知识、体系化内容。长期迭代，持续沉淀。

## 使用规则

1. **一主题一笔记** — 不重复建，持续迭代
2. **模板驱动** — 使用 knowledge-template 创建
3. **密集链接** — 关联所有上下游知识点
4. **持续更新** — 新内容直接更新原笔记

## 子目录

| 目录 | 用途 |
|------|------|
| `concepts/` | 概念定义、原理讲解 |
| `entities/` | 实体页（人物/公司/模型） |
| `comparisons/` | 对比分析、横评 |
| `queries/` | 查询与问答 |
| `summaries/` | 综合总结 |

## 命名规范

```
【分类】核心知识点
```

示例: `【AI工程】大模型微调基础`
