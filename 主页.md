## 🔧 执行总原则

```
先查本页 → 再打开对应工作流详情 → 匹配三类路径 → 执行 → 归档 → 固化记忆
```

### 三类路径

| 类型 | 说明 |
|------|------|
| **资源来源文件** | 任务所需素材、数据来源 |
| **执行模板文件** | 任务输出固定格式模板 |
| **结果归档文件** | 任务成品、记录、日志存储路径 |

### 四步执行流程

1. **匹配模板与路径** — 获取资源来源、执行模板、归档路径
2. **按流程执行** — 严格遵循既定步骤，不新增不简化不更改
3. **分层沉淀** — 总览写本页，详情写 `03-Knowledge/`
4. **固化记忆** — 定稿流程写入 Hermes 长期记忆
