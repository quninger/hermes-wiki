## 🔧 执行总原则

```
先查本页 → 匹配工作流 → 打开详细执行文件 → 按文件步骤执行 → 归档 → 固化记忆
```

### 三类路径

| 类型 | 说明 |
|------|------|
| **资源来源文件** | 任务所需素材、数据来源 |
| **执行模板文件** | 任务输出固定格式模板（04-Resource/templates/） |
| **结果归档文件** | 任务成品、记录、日志存储路径 |

### 四步执行流程

1. **匹配模板与路径** — 获取三类路径：资源来源 / 执行模板 / 结果归档
2. **按流程执行** — 严格遵循详细执行文件，遇问题先查 03-Knowledge
3. **分层沉淀** — 主页.md←总览 / 工作流详情 / 日志归档
4. **记忆固化** — 写入长期记忆，后续任务优先匹配
