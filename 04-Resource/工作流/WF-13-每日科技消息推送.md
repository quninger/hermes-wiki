# WF-13 每日科技消息推送

## 概述
每日抓取国内外科技站点，筛选有重大价值的AI/科技新闻，生成精简早报推送到知识库。

## 数据来源
数据源文档：`04-Resource/数据源/AI科技消息站点.md`
- 国内：36氪、机器之心、量子位、InfoQ、少数派、V2EX
- 国际：TechCrunch、The Verge、Hacker News、ArXiv AI、GitHub Trending、MIT Tech Review

## 输出模板
推送模板：`04-Resource/templates/tech-news-template.md`
格式：标题 + 摘要 + 站点 + URL

## 执行步骤
1. 读取 `04-Resource/数据源/AI科技消息站点.md` 获取站点列表
2. 用 autocli/bb-browser 抓取各站点最新内容
3. 筛选：重大价值 > 一般资讯 > 过滤花边
4. 按模板生成科技早报，存入 `01-Daily/YYYY-MM-DD-科技早报.md`
5. 可选：推送到 GitHub 或其他渠道

## 筛选规则
- ✅ 重大产品发布、融资、技术突破
- ✅ AI/大模型相关进展
- ✅ 开源项目重要更新
- ❌ 花边八卦、纯营销、重复旧闻

## 关联文件
- 数据源: `04-Resource/数据源/AI科技消息站点.md`
- 模板: `04-Resource/templates/tech-news-template.md`
- 输出: `01-Daily/YYYY-MM-DD-科技早报.md`
