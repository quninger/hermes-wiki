### smart-router 自动路由原理

```
用户任务 → smart-router → 分析任务复杂度
                              │
    ┌─────────────────────────┼─────────────────────────┐
    │                         │                         │
简单任务                  中等任务                   困难任务
    │                         │                         │
    ▼                         ▼                         ▼
glm-4-flash              deepseek-v4-flash         deepseek-v4-pro
(免费)                   (付费)                    (付费)
```

### 调用方式

```bash
# 通过 LiteLLM 智能路由
curl -X POST "http://127.0.0.1:4000/v1/chat/completions" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-litellm-local-hermes" \
  -d '{"model":"smart-router","messages":[{"role":"user","content":"任务"}],"max_tokens":100}'

# 指定具体模型
curl -X POST "http://127.0.0.1:4000/v1/chat/completions" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-litellm-local-hermes" \
  -d '{"model":"glm-4-flash","messages":[{"role":"user","content":"任务"}],"max_tokens":100}'
```

### Hermes CLI 调用

```bash
hermes chat --model smart-router --provider litellm -q "任务" -Q
hermes chat --model glm-4-flash --provider litellm -q "任务" -Q
```

## 💰 成本对比

| 场景 | 旧方案 | 新方案 | 省多少 |
|:-----|:------:|:------:|:------:|
| 并行调研3方案 | 3×mimo | 3×**glm** | ~100% |

| 所有auxiliary | auto→mimo | **glm** | ~100% |
