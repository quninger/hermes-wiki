## 🎯 核心目标

```
用户任务
    │
    ▼

│  LangGraph State Graph              │

│  ② 规划节点 (glm-4-flash 免费)      │
```

## 🏗 五种编排方案

### 方案1：直接执行（分数1-5）
主代理直接做，不委派。适合简单问答、文件读写。

### 方案2：并行批量（分数6-7/10）
delegate_task → glm-4-flash（免费）。适合多维度调研、批量文件处理。

### 方案3：串行分步（分数7-9）
收集 glm-4-flash（免费）→ 实现 deepseek（付费）。适合多文件重构。

### 方案4：MoA多视角（仅分析/评审）
GLM + DeepSeek 参考 → Mimo 聚合。适合架构评审、方案对比。

### 方案5：混合流水线（机械+推理）
execute_code → delegate(glm-4-flash) → 主代理(deepseek)。适合先搜索后分析。
