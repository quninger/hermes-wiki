# WF-05 会话归档

## 概述
将 Hermes 会话中有价值的内容按标准格式归档，沉淀到 Obsidian 知识库。

## 数据来源
无外部数据来源，内容来自 Hermes 本地会话。

## 输出模板
归档文件格式：`04-Resource/templates/archive-template.md`

## 执行步骤
1. 价值预判（免费模型）→ 评级 + 摘要
2. check_archive_new.py --list → 查看已归档
3. check_archive_new.py <标题> <时间> → 判断新内容
4. 高/中价值 → 精炼为「问题→方案→效果」
5. 写入 98-Archive/YYYY-MM-DD 会话标题.md
6. verify_archive.py 验证字数

## 关联文件
- 模板: `04-Resource/templates/archive-template.md`
- 去重脚本: `04-Resource/scripts/check_archive_new.py`
- 字数验证: `04-Resource/scripts/verify_archive.py`
