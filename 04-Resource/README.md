# 🔧 04-Resource 资源库

> 模板、教程、素材、参考资料。工具箱，随时取用。

## 子目录

| 目录 | 用途 |
|------|------|
| `templates/` | 笔记模板（日志/项目/知识） |
| `raw/` | 原始资料（文章/论文/转录） |
| `data/` | 数据文件 |
| `js/` | 脚本工具 |

## 模板列表

- `daily-template.md` — 每日复盘
- `project-template.md` — 项目记录
- `knowledge-template.md` — 知识笔记
